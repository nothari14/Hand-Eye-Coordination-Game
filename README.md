# IMPULSE ⚡

A fast-paced reflex game where hesitation costs you.

## 🎯 Overview
IMPULSE is a browser-based reaction game where players try to click a moving target and drain its resistance before it regenerates. The faster you react, the better your score.

## 🔥 Features
- Dynamic moving target
- Resistance system with regeneration
- Real-time Firebase leaderboard
- Personalized in-game punchlines 😈
- Replay with same user or switch players
- Smooth UI with responsive interactions

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Firebase Realtime Database
- Firebase Hosting

## 🚀 How to Run Locally
1. Clone the repository
2. Open:public/index.html
3. Play directly in your browser
